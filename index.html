Comment ça marche ?

  1. L'organisateur crée une session avec un mot de passe secret et saisit les vins
  2. Les dégustateurs rejoignent via le code à 4 lettres, sans mot de passe
  3. Chacun saisit ses réponses vin par vin sur son téléphone
  4. L'organisateur révèle les vins un à un — les points sont calculés à ce moment
  5. Les scores se mettent à jour en temps réel sur tous les appareils

Nouvelle session

Seul celui qui connaît ce mot de passe peut révéler les vins et voir les vraies réponses. Les dégustateurs n'y ont pas accès.

Vins à déguster

Les réponses (producteur, appellation, millésime) sont stockées séparément et ne sont jamais visibles par les dégustateurs.

Rejoindre

Sans mot de passe → dégustateur. Avec le bon mot de passe → organisateur (accès à la révélation et aux vraies réponses).
